# HandFit: Automated Prosthetic Fitting Tool for Blender

This repository contains the source code for the [HandFit project website](https://marcosncosta1.github.io/handfit/).

For the Blender add-on source code, see the [main repository](https://github.com/marcosncosta1/prosthetic-fitter-addon).

## About

HandFit is a free Blender add-on that automates the fitting of 3D-printed prosthetic hands to patient-specific anatomy. It replaces manual CAD workflows with a simple, five-minute process accessible to non-technical users.

**Key Features:**
- Landmark-based alignment
- Anisotropic scaling (independent wrist width and palm length adjustment)
- Socket conformation with millimeter-precise offset control
- Pre-configured Master Model library for all Kinetic Hand variants
